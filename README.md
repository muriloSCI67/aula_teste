\# Primeira Atividade com Git



\## 📌 Objetivo



Esta atividade tem como objetivo aprender as configurações iniciais do Git e entender os primeiros passos para utilizar o controle de versão em projetos.



\## ⚙️ Configurações Iniciais do Git



Antes de começar a utilizar o Git, é necessário configurar algumas informações básicas do usuário.



\### Configurar nome de usuário



```bash

git config --global user.name "Seu Nome"

```



\### Configurar e-mail



```bash

git config --global user.email "seuemail@exemplo.com"

```



\### Verificar configurações



Para conferir todas as configurações feitas no Git:



```bash

git config --list

```



\## 📂 Inicializando um Repositório



Para iniciar um repositório Git em uma pasta do projeto:



```bash

git init

```



\## ➕ Adicionando arquivos



Para adicionar arquivos ao controle de versão:



```bash

git add .

```



\## 💾 Criando um commit



Para salvar as alterações feitas no repositório:



```bash

git commit -m "Primeiro commit"

```



\## 🚀 Conclusão



Com essas configurações básicas, já é possível começar a utilizar o Git para versionar arquivos e acompanhar mudanças em projetos.
